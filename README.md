# CipherDrop

Secure file sharing system built with ASP.NET Core and Clean Architecture.

## Features

- File upload with encryption
- Secure download links
- Expiry + download limits
- JWT authentication
- Audit logging
- File history per user

## Tech Stack

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Vanilla JS frontend

## Setup

1. Clone repo
2. Add your connection string in appsettings.Development.json
3. Run API
4. Run Client
